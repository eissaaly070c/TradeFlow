📚 دليل النشر الشامل — خطوة بخطوة
المرحلة 1: الإعداد (Remix IDE)
  1. افتح remix.ethereum.org من المتصفح
  2. في قائمة الملفات، اضغط New File
  3. سمّ الملف PaymentGateway.sol
  4. الصق كود العقد الأول من الملف المرفق
  5. كرر لـ InvoiceToken.sol و CreditScore.sol
المرحلة 2: ربط MetaMask بـ Polygon Amoy
  1. افتح MetaMask ← Settings ← Networks ← Add Network
  2. Network Name: Polygon Amoy Testnet
  3. RPC URL: https://rpc-amoy.polygon.technology
  4. Chain ID: 80002
  5. Currency Symbol: MATIC
  6. احصل على رصيد مجاني: faucet.polygon.technology
المرحلة 3: تجميع ونشر العقود
  1. في Remix اضغط على أيقونة Solidity Compiler (⚙️)
  2. اختر الإصدار 0.8.20
  3. اضغط Compile PaymentGateway.sol
  4. انتقل لـ Deploy & Run (الصاروخ 🚀)
  5. في Environment اختر Injected Provider - MetaMask
  6. انشر CreditScore أولاً ← احفظ عنوانه
  7. انشر InvoiceToken ← احفظ عنوانه
  8. انشر PaymentGateway مع عنوان USDC testnet كـ parameter
المرحلة 4: ربط العقود ببعض
  1. في CreditScore: استدعِ setPaymentGateway(عنوان_PaymentGateway)
  2. في CreditScore: استدعِ setInvoiceToken(عنوان_InvoiceToken)
  3. اختبر: في PaymentGateway استدعِ registerMerchant بعنوانك
